# HelpDesk Lite 

A **full-stack support ticket management system** built for hackathons with modern React + TailwindCSS frontend and Node.js + MongoDB Atlas backend. Users create tickets, agents track progress, and every issue gets resolved efficiently.

##  Features

- ✅ **Real-time Ticket Dashboard** - Live stats, search, filters
- ✅ **Beautiful UI** - TailwindCSS gradients, animations, responsive
- ✅ **Full CRUD Operations** - Create, Read, Update, Delete tickets
- ✅ **Ticket Numbering** - Auto-generated `#HD-1234` format
- ✅ **Status Tracking** - Open → In Progress → Resolved workflow
- ✅ **Priority Management** - Low/Medium/High with color coding
- ✅ **Search & Filter** - By title, description, status, priority
- ✅ **Mobile Responsive** - Perfect on all devices
- ✅ **Production Ready** - Error handling, loading states, validation

##  Tech Stack

| Frontend | Backend | Database | Tools |
|----------|---------|----------|-------|
| React 18 | Node.js 18 | MongoDB Atlas | Vite |
| TypeScript | Express.js | Mongoose | TailwindCSS 3 |
| React Router | CORS | Free Tier | Axios |
